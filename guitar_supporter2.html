Guitar Supporter2: Increase Accuracy of Hands Model

Home Blog Machine/Deep Learning

Guitar Fingering Recognition Model

I. 서론

지난번 언구 "Shall We Play?: 좋아하는 음악을 연주하고 싶어하는 모든 사람들에게 드리는 선물"에 이어서 나온 후속 연구이다. 지난번 연구에서의 한계점 중 하나인 일반적으로 사람들이 쓰는 모든 코드를 맞출 수 없다는 점과 정확도를 85%에서 더 높이기 위해 기타운지모델을 심층적으로 연구하여 새롭게 발굴해보기로 했다. 더 나아가 가능한 선에서 정밀한 손 모델이 만들어질 수 있다면 만들어보기로 했다.

이번에도 코드를 수집함에 있어서 지난번 연구와 유사한 방식을 활용하기로 했다. 사람들이 흔하게 쓰는 35종의 코드를 YouTube를 활용해서 수집하기로 했다. 이떄, 20,000회 이상의 조회수를 기록한 유튜브 영상을 선택하였다. 이는 데이터의 품질, 정확성, 일반성과 대중성을 만족시키기 위함이였다.

II. 연구 진행과정
1. 수집 및 전처리

수집을 진행할 때 다음과 같은 형태로 수집하였다.

지난번과 마찬가지로 데이터를 200*200 사이즈로 수집하였다. 또한 Media Pipe Model의 인식 때문에 color mode를 'gray scale'로 활용하였다.

이 방식을 활용하여 35개의 기타 코드에 대하여 각각 6000장, 총 21만장의 이미지 데이터를 직접 수집하였다.

수집을 진행하는 과정에서 위의 사진처럼 정상적으로 손이 나오는 경우만이 아닌 다음과 같은 사진이 나오기도 했었다.

가장 적합한 딥러닝 모델을 찾기 위해 다양한 딥러닝 모델을 epoch와 batch를 바꿔가며 적용하였다.

2. 딥러닝 모델에 적용
ㄱ. 기초 CNN 모델

먼저, 딥러닝 모델의 가장 기초가 되는 CNN 모델을 적용하였다.

위 사진에서 노란색으로 강조해둔 것은 layer의 개수이다. 기본값으로 첫번째 Convolution filter을 32로 설정, 두번째 Convolution filter을 64로 설정하였다. maxpooling에서 pool_size와 strides를 처음에는 (2, 2), 두 번째를 (5, 5)로 설정해 빠르게 특징을 잡아서 정확도를 표현할 수 있게 하려 하였다.

또한, 사진에서는 보이지 않지만 이 layer의 개수를 변화해가며 가장 정확도가 높은 경우를 찾으려고 했다. 정확도는 뒤에 언급하고자 한다.

처음에는 21만장으로는 부족하다고 여겨 이미지를 증폭시키고자 augmentation을 실시하였으나, 두가지 경우에 대한 정확도를 비교한 결과 비슷했기 때문에 나중에는 augmentation을 뻈다. 수집한 코드의 경우 8:2의 비율로 train, test data를 나누었다.

또한 위 그림에서 표시한 epoch와 batch 사이즈를 변형시켜가며 분석을 진행하였다. 다음은 각 batch와 epoch에 따른 결과이다.

Epoch: 5, Batch: 64 => Accuracy: 60.28%

Epoch: 10, Batch: 64 => Accuracy: 62.89%

Epoch: 5, Batch: 32 => Accuracy: 55.27%

Epoch: 10, Batch: 32 => Accuracy: 65.63%

이 방법으로는 정확도의 최댓값이 65.63%가 나왔다. layer 수를 변형시켜가면서 했을 때의 정확도는 다음 표와 같이 나타났다.

이 때, layer이 3개일 때 가장 정확도가 높게 나왔다. 이에 layer=3일 때 기준으로 batch와 epoch를 변형시켜가면서 정확도를 확인해보았다. 또한 image 사이즈를 축소했을 때 비슷한 정확도를 보였으므로 속도를 높이기 위해 이미지 사이즈를 100*100으로 줄여서 진행하였다. 결과는 다음과 같다.

직접 모델을 만들어서 가장 최적의 모델을 만들기에는 한계가 명확했다. 정확도가 70%가 넘지 않았기 때문이다. 이에, 다른 사람들이 만든 모델을 활용하기로 하였다.

ㄴ. VGG19 모델

VGG19 모델을 적용해보기로 하였다. 다음은 VGG모델의 원리를 직관적으로 보여주는 그림이다.

위와 같은 방식으로 batch와 epoch를 변형시켜가면서 적용시켜본 결과는 정확도는 다음과 같았다.

최고의 정확도가 약 63%로 기본적인 모델보다 정확도가 낮은 결과가 나타났다. VGG모델의 경우 이미지 사이즈를 축소시키면 도리어 정확도가 낮게 나올 수 있다는 연구결과를 확인하고 다시 이미지 사이즈를 200*200으로 바꾼 후 진행하였다.

지금까지 해본 실험 결과 중 가장 높은 정확도인 69.34%의 정확도를 보였다. 하지만 이것만으로는 정확도가 높다고 할 수 없으므로 다른 모델을 사용해보기로 하였다.

ㄷ. Resnet 모델

다음은 resnet 모델을 활용해서 학습을 시켜보았다. 위와 비슷한 방식으로 실험한 결과는 다음과 같았다.

이 때, epoch 10, batch 64일 때 제일 높은 정확도인 75.02%가 나온다. 다른 모델에 비해 정확도가 높은 이유는 깊은 layer에 따른 gradient vanishing 문제를 해결할 수 있는 resnet의 특성이 도움을 준 것이라고 생각한다.

하지만 정확도가 여전히 80%가 넘지 못했기 때문에 다른 모델을 통해 정확도를 상승을 더 시켜보고자 densenet 모델도 적용해보기로 했다.

ㄹ. DenseNet 모델

마지막으로 densenet 모델을 활용해서 학습을 시켜보았다. 앞과 동일한 방식으로 실험한 결과는 다음과 같았다.

이 모델로 분석을 진행하였을 때에는 어떤 batch와 epoch를 설정하건 상관없이 약 30%의 균등한 오류율이 나타났다.

ㅁ. 다른 관점에서의 접근

모델을 단순히 바꾼다고 정확도를 높일 수 없다는 것을 발견하고, 기본적인 CNN 모델로 해결을 할 수 없음을 발견하고, filter 값을 변형해보기로 하였다. 다음은 여러 방식으로 filter을 변화시키며 분석을 진행한 결과이다.

여러 방식으로 접근을 해보았지만 최종적으로 가장 높은 정확도는 70.07%였다. 이에 필터 또한 정확도에 영향을 미치기는 하나, 이 연구에서는 한계가 있다고 판단하였다.

III. 결과 및 한계
1. 결과

여러가지 모델과 다양한 epoch, batch, filter을 활용해 딥러닝을 진행한 최종적인 결과는 다음과 같다.

가장 높은 정확도를 가진 모델은 ResNet 모델로 약 75%까지의 정확도를 보였다. 다른 분석을의 경우 최대 약 70%의 정확도를 보였다. 하지만 정확도가 80%를 넘기지 않는 상황이기 떄문에 효용성은 없다고 볼 수 있다.

2. 한계점

한계점은 총 4가지가 존재한다. 첫번쨰는 컬러 이미지가 아닌 흑백 이미지로 이루어진 데이터셋이란 점이다. 손 인식 및 분석과 같은 섬세한 인식이 필요한 경우 컬러 이미지가 이번 연구에서 진행한 흑백 이미지보다 더 인식을 잘 할 수 있다고 생각했다. 두번째는 데이터 수집의 한계이다. 처음 raw data를 수집할 때 컬러로 할 방법이 있었더라면 더 좋은 결과가 나왔을 수 있다고 생각한다. 세번째는 딥러닝 기술 자체의 한계이다. 지금 CNN 딥러닝 기술은 대개 차원을 기준으로 하는 것이여서 모양이 아닌 컬러에 최적화된 것으로 보였다. 만약에 최적화하는 다른 방법이 있었더라면 더 좋은 정확도가 나타나지 않았을까 예상한다. 마지막은 효율성 문제다. data 수집 과정에서 image size를 200*200보다 더 큰 사이즈로 수집했거나, 이미지 자체를 더 확대할 방법이 있었다면 정확도를 올렸을 수 있을 것이다.

IIII. 결론

여러가지 딥러닝 모델을 적용한 결과, 정확도가 높게 나올 수 있는 모델은 존재하지 않았다. 한계점에서 말했던 문제들 중 최소 한가지 이상 해결하지 않으면 정확도가 올라가기 쉽지 않을 것으로 보인다.
